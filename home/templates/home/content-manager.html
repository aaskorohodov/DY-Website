Content-Manager

I have started with a very poorly-designed web-site with randomly made content. This is how it was:

So i made several improvements!

You may not be too familiar with what content-managers doing, when having a huge amount of images to process, so let me explain, focusing specifically on images:

  • Standardization: all images got consistent style
  • Brightness: all images become about the same brightness
  • Positioning: all items got the same angle
  • Watermark: I made it less visible and set fixed size for it
  • Colors: color-balance, hue and white-balance got standardized

Take a look at this section. It might not be much, but there is a common style in colors, fonts and elements:

I like standardization and experimenting! So I was trying a lot to find a style, that fits great for this products, clients and company in general. During my work there, I created style-guides, that were describing all the possible products, and all of the aspect about ho to:

  • Take photos: which products should have which angles
  • Set lightning: how to set light in a studio
  • Camera: what kind of camera and lenses to use
  • Description: how should product's descriptions be written
  • Tools: which software to use, to handle photo-editing
  • Adds and news: how to write news, adds and any other texts

Being passion about the work I was doing let us, at that time, to become the best Vape-Shop on the market, with the best content that was admired by many-many clients. We were expanding, so was my department too, and at the end I got several colleagues to teach how to handle the content.
